# Humanex v4.0

Humanex is a PyQt5 desktop application that combines a graphical user interface with Playwright browser automation. The application allows users to configure browser sessions, load proxies, import user agents and cookies, manage target URLs, and monitor session activity through a built-in logging interface.

## Features

- PyQt5-based desktop GUI
- Playwright browser automation
- Multiple target URL management
- Configurable session duration per URL
- Proxy import and management
- User-Agent import and rotation
- Cookie import (Netscape/JSON formats)
- Android/Desktop traffic distribution settings
- Multi-threaded session execution
- Referral source selection
- Google keyword search workflow
- Session logging and monitoring
- License activation window
- Automatic version checking and update support
